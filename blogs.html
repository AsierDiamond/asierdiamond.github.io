Hello dear readers!

We are all aware on the... let's just say controversial decisions of Microsoft with the Windows Operating System. It feels like privacy Isn't an option anymore, and that is just lame and sad.

Microsoft has also bloated Windows 11 to such point that millions of devices that could perfectly run Windows 10, can't even run Windows 11. And believe me, some of these machines are perfectly able to run it, but Microsoft's Strict list of 'Minimum requirements' Say otherwise

This didn't matter for most of the time, Windows 10 was still supported... Until it wasn't

Shockers, now millions of computers are E-waste overnight. Tragic. What now?

My Laptop was somehow capable of running Windows 11, it is an HP Laptop, 15s-fq0 model with 4 GB of ram. Not bad, right? ... right? Wrong.

Let me tell you, when i first opened the Task manager on this thing I was shocked, the sheer amount of Background processes that were running at the same time were colossal. It was consuming half of the memory on idle. I had enough.

I started with Linux Mint, A very Userfriendly Linux Distribution, it is Ubuntu based and it has various flavours, I chose XFCE. A memory-wise flavour.
I installed it, and it worked first time. I purged Windows and it installed in 10 minutes, when I rebooted my computer, no terminal, no grub, just Linux.

I find it funny that people see Linux as a scary, hackery OS, but as I said before, anyone can install it in 20 minutes.

Around 2 weeks later, I was going to install Arch, but there was a Malware attack to its repository, so it was maybe not the best time. So I installed Fedora Linux.

Fedora's Installation was also pretty straightforward, and my god there are sooo many flavours, GNOME, KDE, XFCE, and also, one that caught my attention. LXDE!

This DE was the pinnacle of optimization, I was amazed, it was one of the few times I saw my laptop performing so well.

I stayed here for some time, Until the AUR issue was (mostly?) fixed. So, it was time.

Arch Linux Is one of the last Linux Distributions that you will use if you want User-Friendlyness, those with terminalphobia wont like it.

I love it because it's lightweight, optimized and you have full control of the system, you start with a basic shell, and start building your way up. Getting closer every line you write

It took me a lot of troubleshooting but now I have an LXDE desktop in my laptop that barely consumes any memory, and that is, in fact, where im writting this now!.

So, is it worth it?

Yes, totally, switching from Windows to Linux is one of the smartest decisions you can do when using a computer, nowadays it is completely feasible and easy.

Thats all, cheerio dear Readers!
